# Revenue NSW Assessment

This project automates testing for Revenue NSW using Playwright and Jenkins.

## Project Structure

- `features/` - Contains test feature files for API and GUI tests
- `pages/` - Page object models for Playwright
- `steps/` - Step definitions for API and GUI tests
- `playwright-report/` - Test reports generated by Playwright
- `Jenkinsfile` - Jenkins pipeline configuration
- `package.json` - Project dependencies and scripts

## Running Tests

### Locally

1. Install dependencies:
   ```
   npm install
   ```

```
2. Install Playwright browsers:
```

npx playwright install --with-deps

```
3. Run all tests:
```

npm run test

```
4. Run only GUI tests:
```

npx playwright test features/@gui

```
5. Run only API tests:
```

npx playwright test features/@api

```

### In Jenkins
- The pipeline is parameterized. Select `all`, `ui`, or `api` when starting the job to run the desired tests.
- The Playwright HTML report will be published as a build artifact.
